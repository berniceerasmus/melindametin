Grammar — The 4 Spelling Rules

When to use -ed, -d, double letter, or -ied

8 min▼

✅ Rule 1 — Most verbs: just add -ed

watch→watched · play→played · walk→walked · clean→cleaned
🇹🇷Çoğu fiil için: sona -ed ekle. En temel kural!

✅ Rule 2 — Ends in -e: just add -d

If the verb already ends in -e, add only -d. Don't double the -e!

live→lived · love→loved · arrive→arrived · dance→danced · decide→decided
✗ liveed · loveed — WRONG!
🇹🇷Fiil -e ile bitiyorsa sadece -d ekle. "live"→"lived" (liveed değil!)

✅ Rule 3 — Short CVC verb: double last letter + -ed

CVC = Consonant + Vowel + Consonant (1 syllable). Double the final consonant!

stop→stopped · plan→planned · drop→dropped · chat→chatted
✗ stoped · planed — WRONG! Must double!
🇹🇷Kısa fiiller (1 hece, ünsüz-sesli-ünsüz): son harfi çift yaz + -ed. "stop"→"stopped"

✅ Rule 4 — Consonant + y: change y → ied

If a verb ends in consonant + y, change y → i then add -ed. But vowel+y → just add -ed!

try→tried · study→studied · carry→carried · worry→worried
play→played · stay→stayed (vowel before y — just add -ed!)
✗ tryed · studyed — WRONG!
🇹🇷Ünsüz + y ile biten fiiller: y→ied. "try"→"tried". Ama sesli+y ise sadece -ed: "play"→"played"
🧠
Quick Summary: Rule 1: +ed · Rule 2: ends -e → +d · Rule 3: CVC (1 syllable) → double+ed · Rule 4: consonant+y → ied
